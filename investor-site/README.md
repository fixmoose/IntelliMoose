# IntelliMoose Investor Site

## Overview

This is a comprehensive investor-focused landing page showcasing the IntelliMoose platform vision, market opportunity, and investment details.

## What's Included

### Sections

1. **Hero** - The problem we're solving and why edge AI matters
2. **Market Opportunity** - $500B+ TAM across 5 verticals with detailed breakdown
3. **Platform Editions** - All 5 product editions (Trades, Security, Fleet, Defense, Healthcare)
4. **Competitive Landscape** - Why we win vs. cloud AI, our moats, and "Why Now"
5. **Tech Architecture** - Our technical stack and advantages
6. **Business Model** - Revenue streams, unit economics, 5-year projections
7. **Roadmap** - Quarterly milestones and risk mitigation
8. **Team & Vision** - Founding team, hiring needs, advisors
9. **Investment** - Seed round details, use of funds, why invest

### Key Numbers Featured

- **TAM:** $500B+ across trades, security, fleet, defense, healthcare
- **Raise:** $1.5M - $2M seed round
- **Valuation Cap:** $10-12M
- **Year 5 Revenue:** $280M projected
- **Gross Margins:** 50-65%

## Running the Site

```bash
# Install dependencies
npm install

# Run development server (port 5174)
npm run dev

# Build for production
npm run build
```

## Customization Needed

Before sharing with investors, update:

1. **TeamVision.jsx** - Add your name, bio, and background
2. **Investment.jsx** - Add your contact email, LinkedIn, Calendly link
3. **Add a logo** - Replace /vite.svg with your logo
4. **Add product images** - Real photos of prototype if available

## Design Philosophy

- **Dark, premium aesthetic** - Conveys seriousness and technical depth
- **Data-driven** - Lots of numbers, projections, TAM breakdowns
- **Platform-first messaging** - Emphasizes horizontal opportunity, not single product
- **Investor-friendly** - Clear ask, use of funds, exit paths

## Tech Stack

- React 18
- Vite 7
- CSS (no framework - custom styling)
- React Icons
